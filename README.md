# Spotify Clone

Minimal Spotify-like player built with Next.js 15, TypeScript, Tailwind CSS, Three.js and Framer Motion.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

## Features

- Responsive layout with sidebar and player bar
- Sample song list with basic playback
- Three.js rotating cube visualizer
- Framer Motion animations for smooth interactions

## Deployment

This project is compatible with Vercel. Simply push the repository and import it into your Vercel dashboard.
